        - This example shows how to create a various types of graphs, as well as a pie chart, with the [Flot plugin](https://github.com/jaxon-php/jaxon-flot).

A plot can display multiple graphs, and the series for each graph can be defined either with a list of points, or with a Javascript function.

The multiple types example is the same as [https://www.flotcharts.org/flot/examples/series-types/index.html](https://www.flotcharts.org/flot/examples/series-types/index.html).
The multiple axes example is the same as [https://www.flotcharts.org/flot/examples/axes-interacting/index.html](https://www.flotcharts.org/flot/examples/axes-interacting/index.html).
The pie chart example is the same as [https://www.flotcharts.org/flot/examples/series-pie/index.html](https://www.flotcharts.org/flot/examples/series-pie/index.html).
