Cet exemple montre l'utilisation de l'autoloader de Composer.

Par défaut, la librairie Jaxon enregistre tous les répertoires qui ont un namespace avec l'autoloader `PSR-4`, et tous les répertoires sans namespace avec l'autoloader `classmap`.
